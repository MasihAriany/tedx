مسابقه طراحی لوگو

طرح خود را برای مسابقه طراحی لوگوی ما ارسال کنید.
یک صندلی بصورت رایگان در تدکس دانشگاه شیراز برای برنده رزرو خواهد بود.

    لطفا این قوانین را بادقت مطالعه کنید
    اگر هرکدام از قوانین در طرحی رعایت نشود، طرح از مسابقه حذف خواهد شد.

English دانلود فایل توضیحات

طرح ارسالی باید شامل لوگوی اصلی ما باشد (TEDxShirazUniversity)
بعد از ثبت نام در مسابقه، فایلهای لازم برای شما ارسال خواهد شد.

از نرم افزاری برای طراحی استفاده شود که بصورت برداری بوده و نه Bitmap
درنتیجه، از نرم افزراهایی شبیه Corel استفاده شود و نه برنامه هایی شبیه Photoshop.

رزولوشن فایل خروجی میبایست دقیقا 300dpi باشد.
لطفا از فرستادن کیفیت دیگری خودداری فرمایید.


تنها رنگهای قابل استفاده قرمز، سفید و مشکی میباشند. لطفاً توضیحات زیر را بادقت مطالعه فرمایید.
مشخصات رنگ سفید:
(Hex: #FFFFFF - RGB: 255, 255, 255 - CMYK: 0, 0, 0, 0 - HSV: 0, 0, 100)
و برای قرمز و مشکلی تنها رنگهای قابل مشاهده در تصویر مجاز به استفاده میباشند.
لطفا در نظر داشته باشید که هیچ درجه ی دیگری از این رنگها مجاز نمیباشند.


تمام طرح میبایست در یک واحد ( یک شکل فرضی ) اندازه شود. بطور مثال یک دایره یا مستطیل فرضی را درنظر بگیرید.

موضوع طرح ارسالی باید نمایانگر زمینه ی کنفرانس یا دانشگاه شیراز باشد.
برای آشنایی با طرحهای دیگر کنفرانسها اینجا را مشاهده فرمایید.

فرمت خروجی میبایست PNG باشد. لطفا از فرستادن هر فرمت دیگر خودداری فرمایید.

توجه داشته باشید که طرح میبایست منحصرا متعلق به خودتان بوده و برای اولین بار طراحی شده باشد
در صورت برنده شدن، فایلهای پروژه از شما تحویل خواهد گرفته شد.
هرگونه نسخه برداری از طرحهای دیگر باعث حذف شما از مسابقه میشود.

درصورت داشتن هرگونه پرسش، با ما تماس بگیرید.
